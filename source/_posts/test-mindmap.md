# 思维导图演示

这是一篇演示 Markdown 转换为思维导图的测试文章。

## 使用 hexo-markmap 插件

在 Hexo 博客中，可以使用 `{% markmap %}` 标签来创建交互式思维导图。

{% markmap %}
# 个人技术栈
## 前端技术
- HTML/CSS
- JavaScript/TypeScript
- React
- Vue
- Node.js
## 后端技术
- Python
- Go
- MySQL
- MongoDB
## DevOps
- Docker
- Git
- CI/CD
{% endmarkmap %}

## 思维导图特点

- ✅ 支持 Markdown 语法
- ✅ 可交互（缩放、拖动）
- ✅ 可折叠/展开节点
- ✅ 支持链接和格式化文本

## 使用方法

在文章中使用以下语法：

```
{% markmap %}
# 主题
## 分支1
- 子节点1
- 子节点2
## 分支2
- 子节点3
{% endmarkmap %}
```

这样就可以将 Markdown 列表转换为漂亮的思维导图了！
